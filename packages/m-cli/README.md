# M CLI
`M CLI`是一个伴随组件库而生的脚手架工具，它有以下特性：

- 创建组件库模板
- 创建组件库入口文件

# 快速开始
## 启动服务
```shell
pnpm --filter @m-f2e/m-cli dev
```
## 创建组件库模板
```shell
pnpm --filter @m-f2e/m-cli create
```
## 构建
```shell
pnpm --filter @m-f2e/m-cli build
```
## 命令
- create 创建组件库模板
- build 构建组件库
